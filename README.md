# Java Mini Projects

This repository contains beginner-to-intermediate Java projects created while learning core Java concepts and object-oriented programming.

## Concepts Used
- OOPs
- Classes and Objects
- Constructors
- Methods
- ArrayList
- Conditional Statements
- Loops

## Projects Included

### 1. Student Management System
A simple Java application to store and display student details using ArrayList and classes.

### 2. Bank Account Simulation
A mini banking system supporting deposit, withdrawal, and balance display operations.

### 3. Employee Salary Calculator
A basic salary calculation program using arithmetic operations and business logic.
